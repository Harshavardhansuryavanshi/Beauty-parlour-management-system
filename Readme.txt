How to run the Beauty Palour Management System (BPMS) Project

1. Download the zip file

2. Extract the file and copy bpms folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name bpmsdb

6. Import bpmsdb.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/bpms (frontend)

Synopsis of beauty parlour management system :

Beauty Parlour Management System
Project Synopsis

1. Abstraction :

The Beauty Parlour Management System is a software solution designed to streamline operations in beauty salons. It manages appointments, customer records, billing, inventory, and employee schedules through a single platform. Developed using Node.js and MySQL, the system enhances service quality and administrative efficiency.

2. Introduction :

In the competitive beauty industry, managing daily activities manually can be time-consuming and error-prone. The Beauty Parlour Management System addresses this by offering a centralized system that automates appointments, tracks client preferences, and maintains staff records efficiently.

3. Scope:

• Customer appointment booking and management
• Staff scheduling and service allocation
• Inventory and product tracking
• Billing and payment processing
• Customer feedback and history maintenance
• Admin dashboard for reporting and control

4. Existing System:

Many beauty parlours still rely on manual registers or basic spreadsheet systems. This method is time-consuming and often leads to scheduling conflicts, missed appointments, and inventory mismanagement.

5. Disadvantages of Existing System :

• High chances of scheduling errors
• No customer history tracking
• Difficulty managing multiple staff and appointments
• Manual billing can lead to calculation mistakes
• No data insights or analytics

6. Proposed System :

The proposed system is a web-based application that offers an interactive interface for managing bookings, customer records, services, staff, and inventory. It automates routine tasks, provides reminders, and maintains a database for all operational data, improving accuracy and customer experience.

7. Advantages :

• Efficient appointment management
• Improved customer satisfaction
• Reduced workload for staff
• Transparent billing system
• Data analytics for better decision making
• Easy access to historical records

8. Hardware Requirements :

• Processor: Intel Core i3 or higher
• RAM: 4 GB minimum
• Hard Disk: 250 GB or more
• Display: Standard 15" Monitor
• Input Devices: Keyboard, Mouse

9. Software Requirements : 

• Operating System: Windows 10 / Linux / macOS
• Backend: Node.js
• Frontend: HTML, CSS, JavaScript
• Database: MySQL
• Tools: Visual Studio Code, XAMPP/WAMP

10. Conclusion :

The Beauty Parlour Management System simplifies business operations and enhances customer service quality. It reduces manual errors, saves time, and enables efficient management of appointments, staff, and inventory. This digital solution is ideal for modern beauty parlours looking to scale their business.
